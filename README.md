# ✅ To-Do List App

Ek powerful aur user-friendly **task management application** jo aapke daily tasks ko organize karne mein madad karta hai. Beautiful UI, smooth animations, aur complete task management features ke saath!

## 📝 Project Description

Yeh ek **fully functional To-Do application** hai jo:
- Tasks add, edit, delete aur complete kar sakta hai
- Two separate sections: "Added Tasks" aur "Completed Tasks"
- Real-time task counters display karta hai
- Local Storage mein data save karta है
- Smooth animations aur transitions ke साथ काम करता है
- Mobile-responsive aur user-friendly interface है

## ✨ Features

### 🎯 Core Functionality
✅ **Add Tasks** - Input field se tasks add करें  
✅ **Edit Tasks** - किसी भी task को edit करके update करें  
✅ **Delete Tasks** - Individual tasks को delete करें  
✅ **Mark Complete** - Checkbox से tasks को complete करें  
✅ **Move Between Sections** - Complete/Incomplete के बीच tasks move करें  
✅ **Clear All** - सभी tasks एक साथ delete करें  
✅ **Task Counter** - Added और Completed tasks की संख्या  
✅ **Data Persistence** - LocalStorage में automatic save  

### 🎨 UI Features
- **Two-Column Layout** - Added Tasks | Completed Tasks
- **Live Counters** - Real-time task count display
- **Smooth Animations** - Fade-in effects aur transitions
- **Responsive Design** - Desktop से mobile तक perfect
- **Interactive Icons** - Edit aur delete buttons
- **Input Validation** - Empty tasks add नहीं हो सकते
- **Vibration Effect** - Empty input पर vibrate animation

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Modern styling, animations, flexbox
- **Vanilla JavaScript** - Complete functionality
- **Font Awesome 7.0.1** - Icons (trash, pen icons)
- **Google Fonts** - Poppins & Roboto typography
- **LocalStorage API** - Data persistence
